# 🚨 Emergency Alert Management System (AlertNet)

A simple web-based emergency alert system built for managing and monitoring emergency incidents in real time.

# ▶️ Run the Project Using Docker

## 1. Clone the Repository

```bash
git clone <your-repository-link>
```

---

## 2. Open the Project Folder

```bash
cd emergency-alert-system
```

---

## 3. Start All Containers

```bash
docker-compose up --build
```
